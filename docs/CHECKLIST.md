# Release checklist

Use this checklist before publishing a new store based on the template.

- [ ] Crear MongoDB.
- [ ] Configurar Cloudinary.
- [ ] Configurar Resend.
- [ ] Configurar dominio.
- [ ] Configurar variables `.env`.
- [ ] Ejecutar `npm install`.
- [ ] Ejecutar `npm run build`.
- [ ] Deploy Render.
- [ ] Verificar login.
- [ ] Verificar admin.
- [ ] Verificar checkout.
- [ ] Verificar subida de imagenes.
- [ ] Verificar envio de emails.
- [ ] Verificar HTTPS.
- [ ] Verificar robots.
- [ ] Verificar sitemap.
